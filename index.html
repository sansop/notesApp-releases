v0.6.0 - 2026-06-07

Added

  • Public release-site publishing workflow for a separate public `sansop/notesApp-releases` repository.
  • Static public download page generator with latest/versioned macOS download links and checksum metadata.
  • Focus area icon choices for Running, Dollar sign, and Entertainment.
  • Dashboard-linked task access for todos attached to a focus area.

Changed

  • Focus-area todo saves now preserve existing focus-area links during partial updates unless explicitly cleared.
  • Dashboard layout is more responsive, with focus-area cards using wider available space.
  • App version synchronized to `0.6.0`.

Fixed

  • Todo focus-area links are less likely to be accidentally cleared by dashboard/detail edits.
  • Public release downloads can now be published without making the private source repository public.

Data And Backup Compatibility

  • No backup schema change.
  • Database/app schema remains version `5`.
  • Backup schema remains version `1`.

Known Caveats

  • Public release publishing requires the `PUBLIC_RELEASES_TOKEN` repository secret and the public `sansop/notesApp-releases` repo.
